### serverless原生心智
1. 我的业务是什么？
2. 做这件事情能不能让我的业务出类拔萃？
3. 如果不能，我为什么要做这件事情而不是让别人来解决这个问题？
4. 在解决业务问题之前没有必要解决技术问题

> 在实践 Serverless 架构时，最重要的心智不是选择哪些流行服务和技术，攻克哪些技术难题，而是时刻铭记在心专注业务逻辑，这样更容易让我们选择合适的技术和服务，明确如何设计应用架构。
